Menu Effect Reference v1.0 </doc>

paneoy.dothome.co.kr

SIDE MENU

toggleClass()를 이용하여 클래스를 추가한 뒤, hasClass()로 클래스 여부를 확인하여 사이드 메뉴를 제작하였습니다. 글씨의 변경은 text() 메서드를 이용했습니다.

See the Pen Menu - Side Menu by hyunseo (@hyunseo418) on CodePen.

TAB MENU1

버튼을 클릭하면 기존의 클래스를 삭제(removeClass())하고, 클릭한 요소에 클래스를 추가(addClass())하여 탭 메뉴를 완성하였습니다.

See the Pen Menu - Tab menu1 by hyunseo (@hyunseo418) on CodePen.

TAB MENU2

접근성을 고려하여 초점 이동을 준수한 웹 표준 탭 메뉴를 완성하였습니다.

See the Pen Menu - Tab menu2 by hyunseo (@hyunseo418) on CodePen.

Show/Hidden Menu

스크롤을 내리면 메뉴를 숨기고, 스크롤을 올리면 메뉴가 나타나게 했습니다.

See the Pen Show/Hidden Menu by hyunseo (@hyunseo418) on CodePen.

Sticky Menu

메뉴가 상단에 오면 그 위치에서 고정되도록 했습니다.

See the Pen Sticky Menu by hyunseo (@hyunseo418) on CodePen.

Dot Menu

우측의 닷 메뉴로 현재 콘텐츠의 위치를 확인할 수 있도록 했습니다.

See the Pen Dot Menu by hyunseo (@hyunseo418) on CodePen.
